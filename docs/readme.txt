Welcome to the Lazarus Modbus library
====================================
The Lazarus Modbus library is a Modbus Class I Lazarus implementation of a Modbus
master and slave over TCP/IP (ModbusTCP). Serial I/O (RTU or ASCII) is not
supported.


Installation
============
Before installing lazarus modbus package, please instal Indy 10 (indylaz) from OPM first.
For TCP Master example, the package TIPEdit is also needed. Install from OPM.

The package is tested with Lazarus 4.0 under Windows 10 and Linux Mint.
